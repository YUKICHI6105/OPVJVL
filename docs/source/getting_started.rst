Getting Started
===============

このセクションでは、OPVJVL計測ソフトウェアを初めて利用する方向けに、アプリケーションの
起動から最初の測定までのクイックスタートを説明します。

概要
----
本ソフトウェアは、太陽電池（OPV）のJV/IV特性、発光素子のIV-輝度（JVL）特性を計測するための
デスクトップアプリケーションです。GUIに **PyQt5/PyQt6**、グラフ描画に **PyQtGraph** を
用いており、測定点ごとにリアルタイムでグラフが更新されます。

主な特徴
^^^^^^^^
* **3つの測定モード**: OPVモード（太陽電池）、JVLモード（発光素子、暗IV測定にも流用可）、
  2ch活用モード（Keithley 2612Bの2チャンネルを活用したモードA/モードB）。
* **モック機器での事前検証**: 実機が手元になくても、開発者モード（``--developer``）で
  起動すればダイオード方程式に基づく擬似データで動作確認ができます。
* **測定の排他制御**: 同一機器を共有するモード同士（例: OPV⇔JVL）は同時実行できないよう
  自動的にロックされ、誤操作を防ぎます。
* **安全な中断**: 「中断」ボタンまたは ``Esc`` キーで、いつでも機器の出力をOFFにしてから
  安全に測定を停止できます。

クイックスタート
------------------

1. **Python仮想環境の準備**
   本システムはPython仮想環境内での動作を前提としています。仮想環境が作成されていない
   場合は、あらかじめ :doc:`installation` を参照してセットアップを完了させてください。

2. **仮想環境の有効化**
   PowerShellなどのターミナルを開き、プロジェクトのルートディレクトリで仮想環境を
   有効にします（開発環境の例）。

   .. code-block:: powershell

      # Windows PowerShellの場合（開発環境: Python 3.13 + PyQt6）
      .venv313\Scripts\Activate.ps1

3. **ソフトウェアの起動**
   以下のコマンドを実行してメインプログラムを起動します。実機がない状態で動作確認したい
   場合は ``--developer`` オプションを付けてモック機器を使用できます。

   .. code-block:: powershell

      # 実機を使用する場合
      .venv313\Scripts\python.exe src\app.py

      # モック機器で動作確認する場合（開発者モード）
      .venv313\Scripts\python.exe src\app.py --developer

4. **初回起動時の画面構成**
   ウィンドウは左右2カラムで構成されます。

   * **左カラム**: 「共通保存設定」（サンプル名・保存先）、「OPVモード」「JVLモード」
     「2ch活用モード」の3タブ、実行ログ表示欄。
   * **右カラム**: 選択中のタブに対応するリアルタイムグラフ表示。

5. **機器設定を確認する**
   メニューバーの「設定(&S)」→「測定の設定...」（``Ctrl+,``）を開き、機器選択
   （Keithley 2400 / 2612B）と接続先（COM/VISAポート）を確認・入力します。
   開発者モードで起動している場合のみ、「モック使用」のチェックが選択可能です。

6. **最初の測定を実行する**
   目的のタブ（OPV/JVL/2ch活用）で電圧掃引条件を入力し、「測定開始」ボタン
   （またはショートカット ``F5``）をクリックします。測定が始まるとグラフが
   リアルタイムに更新され、完了後は指定した保存先へCSVファイルが自動保存されます。

   詳細な操作手順は :doc:`tutorials` を参照してください。
